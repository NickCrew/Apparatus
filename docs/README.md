## 🚀 Quick Start Commands

```bash
# Start everything
cd ~/Developer/apparatus
docker-compose up

# Access services
open http://localhost:8090/dashboard    # Web UI
curl http://localhost:8090/healthz      # API health

# Run a test
curl -X POST http://localhost:8090/api/redteam/autopilot/start \
  -H "Content-Type: application/json" \
  -d '{"target":"http://vuln-web:3000","config":{"interval":2000,"maxIterations":5}}'

# View docs
cat docs/quick-reference.md             # This terminal
open docs/architecture.md               # In editor
open docs/diagrams.md                   # Visual diagrams
```
